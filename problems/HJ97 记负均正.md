# HJ97 记负均正

## 描述

首先输入要输入的整数个数 n，然后输入 n 个整数。输出为 n 个整数中负数的个数，和所有正整数的平均值，结果保留一位小数。

0 即不是正整数，也不是负数，不计入计算。如果没有正数，则平均值为 0。

数据范围：$1 \leq n \leq 2000$，输入的整数都满足 $val \leq 1000$

## 输入描述：

首先输入一个正整数 n，然后输入 n 个整数。

## 输出描述：

输出负数的个数，和所有正整数的平均值。

## 示例1

1. 示例1

```txt
输入：
11
1 2 3 4 5 6 7 8 9 0 -1

输出：
1 5.0
```

2. 示例2

```txt
输入：
3
0 0 0

输出：
0 0.0
```
