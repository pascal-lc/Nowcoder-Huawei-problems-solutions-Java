# HJ101 输入整型数组和排序标识，对其元素按照升序或降序进行排序

## 描述

输入整型数组和排序标识，对其元素按照升序或降序进行排序。

* 数据范围：$1 \leq n \leq 1000$，元素大小满足 $0 \leq val \leq 100000$

## 输入描述：

* 第一行输入数组元素个数
* 第二行输入待排序的数组，每个数用空格隔开
* 第三行输入一个整数 0 或 1。0 代表升序排序，1 代表降序排序

## 输出描述：

输出排好序的数字。

## 示例

1. 示例1

```txt
输入：
8
1 2 4 9 3 55 64 25
0

输出：
1 2 3 4 9 25 55 64
```

2. 示例2

```txt
输入：
5
1 2 3 4 5
1

输出：
5 4 3 2 1
```
