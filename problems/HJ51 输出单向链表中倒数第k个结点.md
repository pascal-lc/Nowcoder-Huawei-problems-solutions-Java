# HJ51 输出单向链表中倒数第k个结点

## 描述

输入一个单向链表，输出该链表中倒数第 k 个结点，链表的倒数第 1 个结点为链表的尾指针。

链表结点定义如下：

```C
struct ListNode
{
    int m_nKey;
    ListNode* m_pNext;
};
```

正常返回倒数第 k 个结点指针，异常返回空指针.

要求：

1. 正序构建链表;
2. 构建后要忘记链表长度。

* 数据范围：链表长度满足：$1 \leq n \leq 1000, \quad k \leq n$，链表中数据满足：$0 \leq val \leq 10000$

本题有多组样例输入。

## 输入描述：

输入说明

1. 输入链表结点个数
2. 输入链表的值
3. 输入 k 的值

## 输出描述：

输出一个整数。

## 示例

1. 示例1

```text
输入：
8
1 2 3 4 5 6 7 8
4

输出：
5
```
