# HJ105 记负均正II

## 描述

输入 n 个整型数，统计其中的负数个数并求所有非负数的平均值，结果保留一位小数，如果没有非负数，则平均值为 0

本题有多组输入数据，输入到文件末尾。

* 数据范围：$1 \leq n \leq 50000$，其中每个数都满足 $\vert val\vert \leq 10^6$

## 输入描述：

输入任意个整数，每行输入一个。

## 输出描述：

输出负数个数以及所有非负数的平均值。

## 示例

1. 示例1

```txt
输入：
-13
-4
-7

输出：
3
0.0
```

2. 示例2

```txt
输入：
-12
1
2

输出：
1
1.5
```
